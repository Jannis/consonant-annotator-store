Consonant Annotator Store Example

This is an example page that can be used to test Annotator with `consonant-annotator-store`. It comes with jQuery and Annotator integrated and assumes Annotator is operating against a local annotator store serving from `http://localhost:9090`.

Steps to get this running

  1. Create a new Consonant Annotator store using:

    consonant-annotator-store setup /tmp/annonator-store

  2. Run the standalone Consonant Annotator store server using:

    consonant-annotator-store run /tmp/annotator-store 9090

  3. Reload this page.

    Add/update/delete annotations.

    Restart the store server and reload this page at will. The annotations should be preserved.
